# SaaS Management API

## 🌟 Descripción
SaaS Management API es un **sistema multi-tenant** desarrollado con **ASP.NET Core 8** y **Entity Framework Core**, diseñado para gestionar múltiples empresas y sus usuarios en un entorno seguro y escalable.  

Actualmente cuenta con los modelos iniciales **Company** y **User**, autenticación JWT, y documentación interactiva mediante **Swagger**.

---

## 🚀 Características Principales

- Multi-tenant: varias empresas pueden registrarse y administrar sus propios usuarios.
- Autenticación JWT para seguridad.
- Modelos iniciales:
  - **Company**: representa cada empresa.
  - **User**: representa los usuarios asociados a cada empresa.
- Documentación interactiva con Swagger.
- Base de datos gestionada mediante **EF Core Migrations**.

---

## 🛠️ Tecnologías

- Backend: **ASP.NET Core 8**
- ORM: **Entity Framework Core**
- Base de datos: **SQL Server**
- Documentación API: **Swagger**
- Seguridad: **JWT Authentication**
